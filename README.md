# MD Bilingual College — School Management System (Frontend)
 
Interface web du système de gestion scolaire conçu pour **MD Bilingual College**.

 
## Stack technique
 
- **React JS** — bibliothèque UI
- **Vite** — bundler / serveur de développement
- **CSS Vanilla** — mise en forme sans framework CSS externe

## Fonctionnalités
 
- Gestion des élèves
- Gestion des classes
- Gestion des enseignants
- Gestion des matières
- Saisie et affichage des notes
- Génération et impression des bulletins de notes
- Historique des impressions des bulletins
- Suivi des paiements de scolarité (statut payé / impayé, blocage d'impression si impayé)
